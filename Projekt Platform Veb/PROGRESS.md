## Как задеплоить бота на Railway (инструкция на следующую сессию)

1. Открыть **railway.app**
2. «Start a New Project» → «Deploy from GitHub repo»
3. Войти через GitHub → выбрать **univers-platform**
4. Root Directory → написать: `bot`
5. Нажать Deploy
6. Проверить в Telegram что бот отвечает
