## Зачем этот сайт и для кого

### Что это

Платформа для вебинаров. Эксперт (коуч, психолог, тренер) проводит живые эфиры — и все записи хранятся в одном красивом месте. Ученики приходят, смотрят, записываются на следующий вебинар.

Это не YouTube-канал и не просто список видео. Это личная платформа эксперта — с его именем, стилем и архивом.

### Первый клиент

Владимир — тренер по отношениям и бизнесу. Мы делаем демо, показываем ему, он видит как это выглядит с его контентом — и покупает.

### Бизнес-модель

Мы создаём шаблон платформы и продаём его экспертам за 3 000 ₽/месяц.
GetCourse (главный конкурент) берёт от 4 000 ₽ и сложно настроить.
Мы — дешевле и проще.

# СТРАНИЦА 1 — ВЕБИНАР

**Что это:** Главная страница. Открывается когда человек заходит на сайт.
**Отвечает на вопрос:** «Когда следующий вебинар и как записаться?»

---

## БЛОК 1 — Шапка (header)

### Как выглядит

Узкая тёмная полоска прилипла к верхней части экрана. Когда прокручиваешь страницу вниз — она остаётся на месте, не уходит. Фон чуть-чуть прозрачный (как будто стекло поверх страницы). Снизу шапки — тонкая золотая линия.

```
┌──────────────────────────────────────────────────────────┐
│  Платформа Веб          Вебинар   Архив   [Записаться ▶] │
└──────────────────────────────────────────────────────────┘
```

### Что написано

- **Логотип слева:** `Платформа Веб`
  - Слово «Платформа» — белым цветом
  - Слово «Веб» — золотым цветом
- **Меню по центру:** `Вебинар` · `Архив`
- **Кнопка справа:** `Записаться` — золотой фон, тёмный текст

### Как работает

- Шапка всегда видна при скролле
- Клик «Вебинар» → показывает страницу 1
- Клик «Архив» → показывает страницу 2
- Клик «Записаться» → открывает Telegram-бот для записи
- На мобильном: меню и кнопка сжимаются, остаётся лого + иконка меню

### Как выглядит

Большой тёмный экран на весь экран. В центре — вся информация о следующем вебинаре. Слева — текст, справа — фото спикера в круглой рамке (пока заглушка). Фон — тёмно-синий с лёгким свечением снизу золотого цвета.

Вверху — маленький бейдж с мигающей точкой. Потом большой заголовок с золотым блеском (как будто по тексту идёт волна света). Дата крупно. Потом таймер обратного отсчёта. Потом кнопка.

│  [ 04 ] дня  [ 12 ] часов  [ 37 ] минут  [ 18 ] секунд │
│   ДНИ          ЧАС           МИН           СЕК           │
│                                                          │
│  [ Получить ссылку в Telegram          ▶ ]               │
│  Ссылка придёт автоматически — ничего вводить не надо   │

### Что написано — точный текст

**Бейдж (маленький, вверху):**
`● Следующий вебинар`
(точка мигает золотым цветом)

**Заголовок (большой, с золотым блеском):**
`Как выстроить отношения, в которых хочется оставаться`

**Подзаголовок:**
`Практический вебинар — без лишней теории, только работающие техники`

`15 мая 2026 · 19:00 МСК`

**Таймер:**
Четыре блока рядом: `04 ДНИ` · `12 ЧАС` · `37 МИН` · `18 СЕК`
Цифры — крупные, белые или золотые. Подписи под ними — маленькие, серые.
Таймер считает в реальном времени (секунды идут вниз).

**Кнопка:**
`Получить ссылку в Telegram ▶`
Большая, золотая, широкая. Высота — 56px, текст — 18px.

**Подпись под кнопкой:**
`Ссылка придёт автоматически — вводить ничего не нужно`
Маленький серый текст.

**Фото спикера (справа):**
Пока — круглая рамка с золотой границей и иконкой человека внутри.
Потом — сюда ставится фото Владимира.

### Как работает кнопка

Нажал → открывается Telegram → там наш бот → бот видит @username → присылает:
«Вы записаны! Ссылка на вебинар придёт за час до начала.»

│  говорите — но не        │  │                          │

│                          │  │                          │

✗ Ищете быстрые советы «как удержать мужчину»
✗ Хотите манипулятивные техники

## БЛОК 2 — Сетка карточек (бенто-сетка)

### Как выглядит

Карточки расположены сеткой. Первая карточка — крупнее остальных (занимает место двух карточек). Остальные — одинаковые, поменьше. Каждая карточка тёмная, с тонкой золотой рамкой.

```
┌─────────────────────────────┬──────────────┬──────────────┐
│                             │              │              │
│   БОЛЬШАЯ КАРТОЧКА          │  карточка 2  │  карточка 3  │
│   (популярная)              │              │              │
│                             │              │              │
├──────────────┬──────────────┼──────────────┴──────────────┤
│              │              │                             │
│  карточка 4  │  карточка 5  │         карточка 6          │
│              │              │                             │
└──────────────┴──────────────┴─────────────────────────────┘
```

На мобильном — карточки в один столбик, одна под другой.

### Как выглядит одна карточка

# ДИЗАЙН — ЦВЕТА, ШРИФТЫ, НАСТРОЕНИЕ

## Цвета

| Где | Цвет | Код |
|---|---|---|
| Фон всей страницы | Очень тёмный синий-чёрный | `#0d0d1a` |
| Фон карточек | Тёмный, чуть светлее фона | `#1a1a2e` |
| Основной текст | Светло-серый (почти белый) | `#e5e7eb` |
| Второстепенный текст (даты, подписи) | Серый | `#9ca3af` |
| Акцент (кнопки, иконки, теги) | Золотой | `#fbbf24` |
| Тёмный вариант золота | Золотисто-коричневый | `#d4a017` |
| Рамки карточек | Золото с прозрачностью | тонкая, еле видна |
| Фон карточек (стеклянный эффект) | Белый с прозрачностью | почти невидимый |

**Правило по цветам:**
Золото — только на важных вещах: кнопки, заголовки, иконки, рамки при наведении.
Всё остальное — тёмное и спокойное. Золото привлекает внимание туда, куда нужно.

## Шрифт

Используем системный шрифт — тот, что стоит на устройстве пользователя.
На iPhone — San Francisco, на Android — Roboto, на Windows — Segoe UI.
Это значит: **никаких внешних шрифтов подключать не нужно** — быстро и везде одинаково хорошо выглядит.

## Настроение и ощущение

- **Дорого, но не пафосно.** Как хороший ресторан — красиво, но без надменности.
- **Современно.** Не корпоративный синий, не устаревший Bootstrap.
- **Тепло.** Золото + тёмный фон = уютно, не холодно.
- **Чисто.** Много пространства между элементами. Ничего лишнего.

## Размеры кнопок

Все кнопки — **минимум 48 пикселей в высоту**.
Это важно для телефона: чтобы палец попадал с первого раза.
Текст на кнопках — минимум 16 пикселей.

# КАК САЙТ ВЫГЛЯДИТ НА ТЕЛЕФОНЕ

На телефоне всё перестраивается в один столбик:
- Три карточки → одна под другой
- Два столбика («для вас / не для вас») → один под другим
- Шапка → логотип слева, кнопка-меню справа
- Таймер → цифры крупнее
- Кнопка «Записаться» → появляется внизу экрана (всегда видна при скролле)
- Видеоплеер → на всю ширину экрана
- Отступы от краёв: 16px минимум с каждой стороны
- Минимальный шрифт: 16px (иначе iPhone сам увеличивает и ломает вёрстку)
