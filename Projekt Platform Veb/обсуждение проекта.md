# Обсуждение проекта — Онлайн-платформа для коучей по отношениям

**Клиент:** Тренер Влад — эксперт по построению отношений, 10+ лет опыта
**Дата:** 11.04.2026
**Статус:** Бесплатно, для кейса

---

## Кто клиент и чем занимается

- Тренер по личным отношениям
- Авторская методика: десятки семинаров, сотни личных консультаций
- Учит: техники коммуникации с мужчинами, флирт, общение в мессенджерах и на сайтах знакомств
- Аудитория: женщины, которые хотят улучшить личную жизнь
- Брендинг: премиум, тёмный фон, золотые акценты

---

## Ответ на промпт

### 1. Что создаём — тип продукта

**Онлайн-школа + вебинарная платформа** на базе Univers Platform

Это не просто лендинг — полноценная экосистема:
- Лендинг с продающим описанием программы
- Вебинары live + запись (хранение видео)
- Личный кабинет ученика
- Продажа курсов (тарифы, оплата)
- AI-ассистент 24/7 (отвечает на вопросы, помогает с выбором программы)
- Telegram-бот (уведомления, ДЗ, поддержка)

---

### 2. Для кого — аудитория

**Первичная:** Женщины 25–45 лет, хотят строить отношения с мужчинами, чувствуют себя неуверенно в общении, используют приложения знакомств

**Вторичная (масштаб платформы):** Любые коучи, психологи, тренеры по отношениям, сексологи — платформа универсальна для этой ниши

---

### 3. Какую проблему решает

| Проблема клиента (Влада) | Как платформа решает |
|---|---|
| Нет единого места для учеников | Личный кабинет — всё в одном |
| Вебинары теряются, нет архива | Запись хранится, доступна по тарифу |
| Ручная продажа консультаций | Тарифы + оплата онлайн |
| Нет ответов 24/7 на вопросы | AI-ассистент всегда онлайн |
| Нет пути ученика (воронка) | Бесплатный урок → тариф → курс |

---

## Лучшая идея — и почему именно она

### Онлайн-школа с вебинарами и путём ученика

**Почему лучше лендинга или бота:**
- Влад уже проводит семинары — значит контент есть
- Записи вебинаров = пассивный доход (продаёт снова и снова)
- Личный кабинет удерживает учеников — они возвращаются
- Один раз создал курс — продаёт без своего участия

**Почему подходит для масштаба:**
Такая же платформа нужна психологам, нутрициологам, коучам по карьере, сексологам — один шаблон, много клиентов.

## Что есть на платформе сейчас и что надо добавить

### Уже есть ✅
- Лендинг (адаптируем под нишу "коучинг/отношения")
- AI-агент (отвечает на вопросы клиентов)
- Форма заявки → уведомление в Telegram
- Страницы для 12 ниш (добавляем "Коучинг/Психология")
- Публичная оферта и политика возврата

### Надо добавить 🔧

#### Приоритет 1 — Вебинары и видео
- **Live-вебинары:** встраиваем через YouTube Live iframe (бесплатно, не нужен отдельный сервис)
- **Запись:** видео загружается на YouTube (скрытая ссылка) или VK Видео → вставляется в личный кабинет
- **Хранилище:** YouTube — рекомендуем. Бесплатно, надёжно, работает в России через VPN или VK Видео как альтернатива

#### Приоритет 2 — Личные кабинеты
- Страница ученика: купленные курсы, прогресс, домашние задания
- Доступ по тарифу (что оплачено — то открыто)
- Реализуется через Supabase Auth (уже в плане)

#### Приоритет 3 — Тарифы и курсы
- Тарифная сетка (например: Базовый / Расширенный / VIP с разбором)
- Продажа курса как отдельного продукта
- Запись на консультацию (платная, через календарь)

#### Приоритет 4 — Путь ученика (воронка)
```
Лендинг → Бесплатный урок (email/Telegram) → Платный тариф → Курс → VIP-консультация
```

#### Приоритет 5 — Что можно добавить к идее Влада
- **Квиз "Какой у вас тип общения с мужчинами?"** → сбор лидов + персональная рекомендация программы
- **Telegram-бот для практики:** ежедневные упражнения по флирту прямо в мессенджере
- **Сертификат** после прохождения курса (автоматически)

## Стек для реализации

| Задача | Инструмент | Стоимость |
|---|---|---|
| Лендинг + страницы | Univers Platform (Vercel) | Бесплатно |

| Оплата | ЮKassa / вручную (карта) | % с оборота |
| AI-ассистент | Claude API | ~$5–20/мес |
| Telegram-бот | Grammy (уже умеем) | Бесплатно |
